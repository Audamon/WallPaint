# WallPaint
 Sistema desenvolvido com React.js para calcular quantidade de latas de tinta necessária para pintar uma sala de quatro paredes.

## Downloads

git clone https://gitlab.com/Audamon/wallpaint

## Instalação 

### `npm i`
 Instala as dependências.

## Rodar

### `npm start`

Roda a aplicação
